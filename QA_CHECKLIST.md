# QA checklist (mobile + desktop)

## Mobile Safari (iPhone)
- Login input: focus does not zoom.
- Chat input: focus does not zoom, field stays visible above keyboard.
- Chat overlay: safe-area bottom padding looks correct.
- Resume: background app then return, countdown stays correct.

## Mobile Chrome (Android)
- Chat input visible when keyboard opens.
- Chat overlay closes and reopens without layout jump.

## Desktop (Chrome/Firefox)
- Resume: switch tab and return, countdown stays correct.
- Chat input size unchanged.
